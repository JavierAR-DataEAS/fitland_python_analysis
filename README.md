## FIT_LAND — Análisis de Negocio con Python

Conjunto de scripts para análisis financiero y de ventas de FIT_LAND,
negocio de ropa deportiva. Feb–Mar 2026.

## Scripts incluidos

| Archivo | Descripción |
|---|---|
| `analisis_ventas.py` | Limpieza de datos y resumen de ventas por mes y categoría |
| `analisis_gastos.py` | Clasificación de gastos: operativos, inversión y retiros |
| `graficas_ventas.py` | 4 gráficas: tendencia diaria, por categoría, día de semana y comparativa mensual |
| `prediccion_abril.py` | Modelo de regresión lineal + promedio móvil para proyectar ventas de abril |
| `dashboard_fitland.py` | Dashboard completo con 4 paneles y predicción integrada |
| `reporte_negocio.py` | Reporte ejecutivo con acciones prioritarias: reposición, stock muerto y márgenes |

## Tecnologías
Python · Pandas · Matplotlib · NumPy · Scikit-learn

## Datos
Los CSVs de ventas, gastos e inventario no se incluyen por privacidad del negocio.
